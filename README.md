# VIA Keymaps

## CIDOO v68

### Layer 0

- Volume & Mute on Knob

![v68 - Layer 0](images/v68-layer0.png)

### Layer 1

- F1 - F12 on top row
- Arrow keys on WASD
- Delete on Backspace & Page Down
- Home / End on < / >, respectively
